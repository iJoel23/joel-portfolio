# Joel Leon — Developer Portfolio

My personal portfolio, built from scratch as a single-page app with a focus on
motion, accessibility, and clean component architecture.

🔗 **Live:** https://ijoel23portfoliov3.netlify.app/

## Tech Stack

- **React** + **TypeScript**
- **Vite** (build tooling)
- **Tailwind CSS** (v4)
- **Motion** (animations)
- **react-i18next** (English / Spanish internationalization)

## Features

- Fully responsive, mobile-first design
- Scroll-triggered animations and a cursor-reactive ambient glow
- Animated project detail sheets (case studies)
- Bilingual (EN/ES) with a smooth language toggle
- Accessibility-first: respects `prefers-reduced-motion`, semantic HTML, keyboard support

## Getting Started

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build
```
